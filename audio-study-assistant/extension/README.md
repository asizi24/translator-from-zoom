# Audio Study Assistant - Chrome Extension

תוסף כרום שמעלה אוטומטית הקלטות אודיו ל-Cloud Run ופותח את התוצאות בטאב חדש.

## תכונות

✅ **העלאה אוטומטית** - מזהה הורדות אודיו ומעלה אוטומטית לשרת  
✅ **פתיחת תוצאות** - פותח טאב חדש עם סיכום וחידון  
✅ **העלאה ידנית** - גרור קבצים ישירות לתוסף  
✅ **הגדרות** - אפשרות לשנות כתובת שרת ולכבות העלאה אוטומטית  

## התקנה

1. פתח Chrome ולך ל: `chrome://extensions/`
2. הפעל "Developer mode" (מצב מפתחים)
3. לחץ "Load unpacked" (טען תוסף לא ארוז)
4. בחר את התיקייה `extension`

## הגדרה

1. לחץ על אייקון התוסף בסרגל הכלים
2. הזן את כתובת ה-Cloud Run שלך
3. לחץ "שמור הגדרות"

## שימוש

### אוטומטי (מומלץ)

1. הורד קובץ אודיו מכל מקום
2. התוסף יזהה את ההורדה אוטומטית
3. הקובץ יועלה ויעובד ברקע
4. טאב חדש ייפתח עם התוצאות

### ידני

1. לחץ על אייקון התוסף
2. גרור קובץ לאזור ההעלאה
3. המתן לעיבוד
4. התוצאות ייפתחו בטאב חדש
